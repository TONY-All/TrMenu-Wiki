# Take Items

## Aliases

* TAKE\(-\)ITEM\(s\)
* REMOVE\(-\)ITEM\(s\)

## Example

```
TAKE-ITEM: material:stone,lore:&bSpecific,amount:3

TAKE-ITEM: material:stone,lore:&bSpecific,amount:3;material:diamond,name:&bMagic Diamond
```
