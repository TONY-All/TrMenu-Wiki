# Delay

{% hint style="info" %}
Difference between **ActionDelay** and **ActionOption.Delay**

When executing **ActionDelay**, other actions stop and wait for the delay

**Option.Delay** is only delay one action
{% endhint %}

## Aliases

* DELAY
* WAIT

## Example

```
DELAY: 20
```
