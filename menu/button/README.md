# Button

## Button Structure

* Button
  * Update Time \(lore, name, materials, placeholders etc.\)
  * Refresh Time \(priority icons\)
  * **Default Icon Display**
  * **Default Icon Actions**
  * Priority Icons
    * pIcon1
      * pIcon1's **Display Part**
      * pIcon1's **Actions Part**

## Icon Structure

* Icon
  * **Display**
    * Name
    * Lore
    * Materials
    * Slots
    * Shiny
    * Amount
    * etc...
  * **Actions**
    * ClickType : ClickaActions

**ClickTypes**

* LEFT
* RIGHT
* SHIFT\_LEFT
* SHIFT\_RIGHT
* MIDDLE
* DOUBLE\_CLICK
* DROP
* CONTROL\_DROP
* ALL
