# 动作

{% hint style="info" %}
动作只有写在 **actions.&lt;交互类型&gt;** 下才有效
{% endhint %}

```yaml
actions:
  TYPE1: ACTION1
  TYPE2:
    - 'ACTION1'
    - 'ACTION2'
```

## 点击类型

* **LEFT**
* **RIGHT**
* **SHIFT\_LEFT**
* **SHIFT\_RIGHT**
* **MIDDLE**
* **DOUBLE\_CLICK**
* **DROP**
* **CONTROL\_DROP**
* **ALL**
